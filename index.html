How to keep it updated

GitHub-friendly workflow
  1. Upload this single HTML file to a GitHub repository, then turn on GitHub Pages in the repository settings.
  2. Open the live page after each matchday and use the form to enter results for any tracked team that played.
  3. After you add or adjust results, click Download match data and save the JSON backup to your computer.
  4. For a permanent public update on GitHub, open the HTML file, find the matchLog array near the bottom, paste in the latest JSON, commit, and push.
  5. If you ever need to change ownership or fix a team name, edit the participants data block near the bottom of this file and commit the update to GitHub.
Simple setup option: keep using the page locally and only update GitHub every few days. The scoreboard recalculates automatically from the saved match list, so you do not need to edit totals by hand.
